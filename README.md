# Pooja Maurya - Portfolio Website

A modern, responsive portfolio website for Pooja Maurya, an aspiring software developer and IT student.

## Features

- **Modern Dark Theme**: Clean, professional design with a dark color scheme
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **3D Effects**: Subtle 3D animations and hover effects for an engaging user experience
- **Smooth Scrolling**: Seamless navigation between sections
- **Interactive Elements**: Animated skill tags, project cards with tilt effects, and more
- **Contact Form**: Functional contact form (placeholder implementation)

## Sections

1. **Home/Hero**: Introduction with call-to-action buttons
2. **About**: Personal background and key statistics
3. **Skills**: Technical skills organized by category
4. **Projects**: Showcase of key projects with descriptions and tech stacks
5. **Resume**: Education, certifications, and coding profiles
6. **Contact**: Contact information and message form

## Technologies Used

- HTML5
- CSS3 (with animations and 3D transforms)
- JavaScript (ES6+)
- Font Awesome for icons

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. The website will load and be fully functional

## Customization

### Personal Information
- Update the hero section with your photo (replace the placeholder image)
- Modify text content in `index.html`
- Update contact information and social links

### Styling
- Colors and themes can be adjusted in `styles.css`
- 3D effects and animations are defined in CSS and JavaScript

### Projects
- Add or modify project cards in the HTML
- Update GitHub and live demo links
- Replace placeholder images with actual project screenshots

### Resume
- Update education details
- Add/remove certifications
- Link to actual coding profiles
